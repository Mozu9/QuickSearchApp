# QuickSearchApp

## 説明
軽量の常駐型検索ランチャーUIアプリ  
A lightweight, stay-on-top search launcher with system tray support.  
- 検索窓にキーワードを入れてEnterで検索  
- .ini編集もしくは右クリックメニューから検索サイトの追加・削除が可能
